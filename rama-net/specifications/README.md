# Specifications

## Net

A non-exhaustive collection of specifications as implemented,
relied upon by rama-net or related to. The interests of this crate are very broad,
and as such it can also reflect upon the wider selection of specifications included here.

### RFCs

#### URI

* [rfc3986.txt](./uri/rfc3986.txt)  
  Defines the syntax of Uniform Resource Identifiers (URIs).

* [rfc3987.txt](./uri/rfc3987.txt)  
  Defines Internationalized Resource Identifiers (IRIs).

#### TLS

* [rfc8446.txt](./tls/rfc8446.txt)  
  Defines TLS 1.3, the modern transport security protocol.

* [rfc5246.txt](./tls/rfc5246.txt)  
  Defines TLS 1.2.

* [rfc6066.txt](./tls/rfc6066.txt)  
  Defines TLS extensions such as Server Name Indication.

* [rfc7301.txt](./tls/rfc7301.txt)  
  Defines ALPN, used to negotiate protocols like HTTP 2 and HTTP 3.

* [rfc6962.txt](./tls/rfc6962.txt)  
  Defines Certificate Transparency for auditing TLS certificates.

* [rfc5280.txt](./tls/rfc5280.txt)  
  Defines the X.509 public key infrastructure and certificate format.

* [rfc5077.txt](./tls/rfc5077.txt)  
  Defines TLS session resumption using session tickets.

* [rfc5705.txt](./tls/rfc5705.txt)  
  Defines keying material exporters for TLS.

#### Misc

* [rfc863.txt](./misc/rfc863.txt)  
  This RFC specifies a standard for the ARPA Internet community.  Hosts on
  the ARPA Internet that choose to implement a Discard Protocol are
  expected to adopt and implement this standard.
