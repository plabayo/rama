🦙 Rama (ラマ)
Modular Service Framework for Rust

Move and Transform Network Packets

Rama is a powerful, modular service framework for Rust that helps you build proxies, servers, and clients with full control over your network stack.

Get Started View on GitHub

🚀 Async-First

Built on Tokio, Rama provides a modern async runtime for high-performance network applications.

🔌 Modular

Pick and choose the components you need. From TCP/UDP to HTTP and TLS, build your perfect stack.

🛡️ Type-Safe

Leverage Rust's type system to build robust and maintainable network services.

🌐 Protocol Support

Support for HTTP/1.1, HTTP/2, TLS, and more. Build proxies, servers, and clients with ease.

📊 Telemetry

Built-in support for tracing and metrics to help you monitor and debug your services.

🔍 Fingerprinting

Advanced fingerprinting capabilities for HTTP and TLS layers to help with emulation.
